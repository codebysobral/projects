## Week 1 Evolution

# Day-01:
I started by learning the concepts of version control (VCS), the
difference between Git and GitHub, and how to install and configure Git.

# Day-02:
I learned the basic concepts of creating and staging files, made my
first commit, and repeated the workflow of creating, staging, and
committing files.

# Day-03:
I learned the differences between the states before and after `git add`
using `git status`, amended a commit message, and created a functional
`.gitignore`.

# Day-04:
I learned how to create new branches, created a new branch and switched
to it. I made changes to a file that were only visible in the new
branch. I also renamed and deleted test branches.

# Day-05:
I learned how to merge two branches, simulated merge conflicts and
resolved them, and visualized the merge history using
`git log --oneline --graph`.

# Day-06:
I connected my local repository to my remote repository for the first
time, performed my first `git push`, made a change on GitHub and
retrieved it using `git pull`, and cloned a repository using
`git clone`.
