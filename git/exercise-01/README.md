## What is Version Control?

Version Control is a system that helps you manage different versions of 
a project. It allows you to track changes, make updates safely, and 
restore previous versions whenever necessary.

## What is the difference between Git and GitHub?

Git is a Version Control System (VCS). It is a tool used to track 
changes and manage different versions of a project.

GitHub is a cloud-based platform that hosts Git repositories. It makes 
it easy to store, organize, share, and collaborate on projects with 
other developers.
