#Today, 6 days ago, I created my GitHub account to upload my local repositories.


*1. Creating an account and creating repositories*
I entered `https://github.com` and clicked on sign up to create
my account. After that, I created my online repositories with the same names
as my local repositories.


*2. Used Commands*
After creating my online repositories, I used the following commands:

`git init` - I used this command to initialize my local repository
and start tracking changes with Git.

`git remote add origin <url-https>` - Used to connect my local repository
to my online repository on GitHub.

`git push -u origin main` - I used this command to send commits from my local repository
to my online repository.

`git pull` - I used this command to update my local repository with changes
from the online repository.
